Open Slider

Universal slider & carousel library. Framework-agnostic core with adapters for 10+ frameworks.

Live demos View on GitHub
Works with Vanilla JS React Vue Svelte Solid Angular Astro Next.js Nuxt Web Component

Quick install

npm install @open-slider/core @open-slider/styles

Basic usage core

import { createCarousel, createSingleRange, createDualRange } from '@open-slider/core';

const carousel = createCarousel({ totalSlides: 4, loop: true });
carousel.next();
console.log(carousel.getState()); // { index: 1, canNext: true, canPrev: true }

const range = createSingleRange({ min: 0, max: 100, step: 5, initialValue: 50 });
range.setValue(75);
console.log(range.getState()); // { value: 75, min: 0, max: 100 }
