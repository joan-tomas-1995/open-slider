Open Slider

Librería universal de sliders/carousel con soporte para Vanilla JS, React, Vue, Svelte, Solid, Angular, Astro, Next y Nuxt.

Instalación rápida

npm install @open-slider/core @open-slider/styles

Uso básico (core)

import { createCarousel } from '@open-slider/core';

const slider = createCarousel({ totalSlides: 4, loop: true });
slider.next();
console.log(slider.getState());
